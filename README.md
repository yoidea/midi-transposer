# midi-transposer

## Install

```
pip install "git+https://github.com/yoidea/midi-transposer"
```

## Usege

```
usage: mtrans [-h] [-a] [-t TRANSPOSE] [-o OUTPUT] source

positional arguments:
  source                source

optional arguments:
  -h, --help            show this help message and exit
  -a, --analyze-key     analyze key
  -t TRANSPOSE, --transpose TRANSPOSE
                        transpose
  -o OUTPUT, --output OUTPUT
                        output
```
